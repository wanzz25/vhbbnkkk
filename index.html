06 Tentang Wanzz Deploy

Tentang Wanzz Deploy

Platform demo deployment bertema cyber, dibangun untuk menunjukkan bagaimana pengalaman "deploy tanpa ribet" bisa terasa — cepat, visual, dan tanpa perlu login.

Apa itu Wanzz Deploy?

Wanzz Deploy adalah antarmuka simulasi deployment yang meniru alur kerja platform seperti Vercel dan Netlify: pilih repository, tekan tombol deploy, lalu pantau prosesnya lewat console log yang berjalan secara langsung. Semua interaksi dirancang terasa instan dan hidup, dari partikel latar belakang sampai animasi tombol.

Bagaimana cara kerjanya?

Tidak ada proses pendaftaran. Pengguna cukup menekan tombol Quick Access untuk langsung masuk ke dashboard. Panel status API menampilkan kondisi koneksi ke GitHub, Vercel, dan Netlify, dan panel Upload Project memungkinkan pengguna menaruh file .html atau .zip langsung dari perangkat mereka untuk dilihat isinya di dalam browser.

Apakah ini gratis?

Ya — Wanzz Deploy 100% gratis digunakan sebagai demo/showcase. Tidak ada biaya, tidak ada akun berbayar, dan tidak ada watermark tersembunyi. Ini adalah proyek open showcase dari Wanzz Dev.

Catatan penting

Wanzz Deploy versi ini berjalan sepenuhnya di sisi browser (client-side). Proses "deploy" pada console log adalah simulasi visual untuk keperluan demo/presentasi, dan proses ekstrak ZIP pada panel Upload Project dilakukan secara lokal di perangkat kamu — tidak ada file yang dikirim ke server manapun.
